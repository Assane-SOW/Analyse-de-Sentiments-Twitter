# Analyse-de-Sentiments-Twitter
Analyse de Sentiments Twitter — Détection de discours haineux


Classification de tweets offensants via Machine Learning, déployé sur AWS.

- 📊 31 962 tweets analysés — Recall : **86%** | Accuracy : **83%**
- 🤖 Modèle : XGBoost (Gradient Boosting) + vectorisation Bag of Words
- ⚖️ Gestion du déséquilibre de classes : Oversampling / Undersampling
- ☁️ Déploiement AWS : SageMaker → Lambda → API Gateway → WebApp
- 🛠️ Stack : Python · Scikit-learn · XGBoost · AWS · GitHub Pages
